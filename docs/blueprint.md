# **App Name**: Laboratorio Creativo Modular

## Core Features:

- Gestión de Equipos y Competencia: Crea y gestiona equipos, asigna nombres editables y visualiza un ranking en tiempo real con un sistema de puntaje manual. Guarda la configuración del equipo y sus resultados.
- Temporizador de Reto Dinámico: Un temporizador configurable con cuenta regresiva visual grande, barra de progreso animada, y controles (iniciar/pausar/reiniciar), que emite una alerta al finalizar.
- Ruleta Animada de Variables: Un spinner animado para la selección aleatoria y destacada de variables configurables (sector, tipo de producto, público objetivo, tecnología).
- Generador de Estrategia AI: Una herramienta de IA potenciada por Genkit y el modelo Gemini, que genera dinámicamente misión y visión profesional, y una propuesta de valor basada en variables de sector, producto, público y diferencial del proyecto, mostrando un resultado editable.
- Exportación PDF y QR: Generación y descarga automática de un PDF con el resumen del equipo, los resultados de la ruleta, la misión y visión generadas por IA y la puntuación. Además, genera códigos QR dinámicos por equipo que incluyen sus variables asignadas y un enlace interno.
- Panel Administrativo Docente: Una interfaz administrativa clara para docentes, que permite crear nuevos retos, reiniciar competencias, editar variables de la ruleta, monitorear el progreso de los equipos, y descargar reportes PDF individuales.

## Style Guidelines:

- Color principal: Púrpura vibrante (#8F22DA) para evocar creatividad y dinamismo, destacando en elementos interactivos y encabezados.
- Color de fondo: Un blanco casi puro con un matiz púrpura sutil (#FDFBFF) para un diseño minimalista y una legibilidad óptima en modo claro.
- Color de acento: Amarillo brillante (#F8E20B) para llamados a la acción, notificaciones importantes y elementos de enfoque.
- Tipografía principal: 'Montserrat' (sans-serif) para todos los elementos de texto, proporcionando un estilo moderno, limpio y profesional. Note: currently only Google Fonts are supported.
- Uso de íconos lineales y minimalistas de la biblioteca 'Lucide React' (o similar) para mantener la estética moderna y clara.
- Diseño responsive basado en una arquitectura modular de componentes, asegurando adaptabilidad a dispositivos móviles y de escritorio con espacios generosos y jerarquía visual clara.
- Microinteracciones sutiles en botones, transiciones fluidas entre secciones y una barra de progreso animada para el temporizador, ofreciendo una experiencia de usuario ágil y con feedback visual.
